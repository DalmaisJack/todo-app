# React + Python ToDo App

フルスタックTODOアプリケーション（React + Flask）

## 🚀 ライブデモ

**[🔗 アプリを試す](https://todo-app-lovat-chi-18.vercel.app)**

## ✨ 機能

- ✅ TODO追加・削除・完了切り替え
- 📊 完了数・総数の統計表示  
- 🎨 美しいUI（グラデーション背景）
- 📱 レスポンシブデザイン

## 🛠️ 技術スタック

- **フロントエンド**: React, CSS3
- **バックエンド**: Python, Flask
- **デプロイ**: Vercel (Frontend) + Railway (Backend)
- **API通信**: Axios, REST API

## 📦 使用方法

1. 新しいタスクを入力欄に記入
2. 「追加」ボタンでTODO追加  
3. ⭕をクリックして完了/未完了を切り替え
4. 🗑️ボタンでTODO削除
